8vo proyecto
titulo : Desarrollo de Sistema de registro y control de liga de beisbol
Descripcion: Desarrollo de un sistema de registro y control para la Asociacion de Umpires del estado Merida
            donde este cuenta con un login de usuarios que tienen funciones como guardar, editar, consultar y eliminar informacion alojada en una base de datos sobre
            los equipos, jugadores y anotaciones  dando resultados de tablas con las mejores posiciones en cada categoria
            El sistema fue realizado con PHP puro desarrollando la base de datos  con su diagrama entidad relacion,
             MYSQL, javascript, AJAX, librerias como bootstrap y para exportar datos en PDF
Enlace a repositorio : https://github.com/KevinRangels/SIBeisball

9vo proyecto
titulo : Desarrollo de Sistema de registro y control a la alcaldia
Descripcion: Desarrollo de un sistema de registro y control para el departamento de 
            recursos humanos de la alcaldia  del estado Merida  donde este cuenta con
            un login de dos tipos de usuarios que depende de este tienen funciones como
             guardar, editar, consultar y eliminar informacion alojada en una base de datos sobre
            la informacion y documentos que deben entregar las personas cuando comienzan a trabar en la alcaldia,
             
            El sistema fue realizado con PHP puro desarrollando la base de datos  con su diagrama
             entidad relacion, MYSQL, javascript, AJAX, librerias como bootstrap, tablas dinamicas
             para la consulta como exportacion en distintos formatos de la informacion y paginacion de los registros
Enlace a repositorio : https://github.com/KevinRangels/SistemaAlcaldiaMerida
